# 팀별 주간 루틴 & 개인 KPI

> 매주 반복하면 습관이 되고, 습관이 되면 성과가 된다.
> 이 문서는 각 팀/개인이 **매주 루틴처럼 지켜야 할 활동**과 **개인별 KPI**를 정리한 가이드입니다.

---

## 설계 원칙

이 루틴은 3가지 관점을 결합해서 설계했습니다.

1. **세일즈 베스트 프랙티스**: HubSpot, Salesforce, Predictable Revenue 등 B2B 세일즈 고성과 팀의 주간 루틴 구조
2. **교육 B2B/B2G 영업 전문가 관점**: 대학/K-12/기업 대상 EdTech 세일즈의 고유한 계절성, 의사결정 구조, 너처링 방식
3. **ESFJ 스타일 관계 유지**: "고객을 사람으로 대하는" 따뜻한 관계 관리를 팀 전체가 체계적으로 반복하는 시스템

---

## 공통 — 전 팀 주간 루틴

모든 팀이 매주 공통으로 지키는 루틴입니다.

| 요일 | 루틴 | 소요 시간 | 설명 |
|------|------|----------|------|
| **월** | 주간 킥오프 | 30분 | 지난 주 핵심 숫자 리뷰 + 이번 주 목표 Top 3 + 막힌 것 공유 |
| **월** | 고객 관계 점검 | 15분 | "이번 주 누구에게 관심을 줄까?" — 2주 이상 연락 안 한 고객, 생일/기념일 체크, 미팅 후속 미완료 건 파악 |
| **수** | 중간 점검 | 15분 | 이번 주 목표 대비 50% 이상 진행됐는지 팀 리드가 확인. 미달 시 우선순위 재조정 |
| **금** | 주간 마감 | 30분 | 파이프라인 업데이트 + 주간 보고 + 다음 주 계획 |
| **금** | 감사 마무리 | 10분 | 이번 주 만난 고객에게 짧은 감사 메시지 발송 (업무 얘기 없이 순수 감사만) |

### 관계 관리 핵심 원칙 (ESFJ 스타일)

모든 팀원이 매주 기억할 5가지:

1. **기억하라**: 고객이 말한 사소한 것을 기록하고, 다음 연락에서 언급한다
2. **먼저 주라**: 용건 없이도 연락하고, 도움 될 자료를 선제적으로 보낸다
3. **일관되게 하라**: 한 번 잘하는 것이 아니라, 매주 반복하는 것이 관계다
4. **진심을 담아라**: 템플릿은 구조일 뿐이다. 한 줄은 반드시 그 사람만을 위한 문장으로
5. **기록하라**: 기억력에 한계가 있으면, 기록이 대신한다. 고객 카드를 항상 업데이트

### 고객 접촉 후 필수 프로토콜

```
미팅 후 1시간 이내  → 감사 문자 발송
미팅 후 24시간 이내 → 미팅 요약 + 약속 자료 이메일 발송
미팅 후 3일 이내    → 보충 자료 추가 발송 ("말씀하신 OO 관련 자료도 함께 보내드립니다")
미팅 후 1주일 뒤    → 후속 체크인 ("보내드린 자료 확인해보셨나요?")
```

# 1팀. 국내 K-12 코들 세일즈팀

**팀 KPI**: 매출 확대 (신규 학교 도입 + 기존 학교 재구매)

## 주간 루틴

| 요일 | 리드 (박기남) | 팀원 (이주현) |
|------|-------------|-------------|
| **월** | 파이프라인 전체 리뷰 + 이주현 님 주간 TODO 지시 + 이번 주 방문 학교 확정 | 주간 TODO 수령 + 이번 주 컨택/방문 대상 정리 |
| **화** | 신규 학교 미팅 (의사결정자 동행) + 핵심 선생님 관계 관리 | 기존 학교 선생님 단독 미팅 2~3건 + 코들 활용법 지원 |
| **수** | 중간 점검 — 이주현 님 진행 상황 확인 + 세일즈 스킬 코칭 (주 1회) | 아웃바운드 컨택 집중 (신규 학교 전화/이메일 10건+) + 데모 세팅 |
| **목** | 학교 현장 방문 2~3곳 + 커리큘럼/연수 기획 | 학교 방문 2~3곳 (단독) + 후속 자료 발송 + 견적서 작성 |
| **금** | 파이프라인 현황 보고 (대표에게) + 다음 주 전략 수립 | 일일 보고 + 선생님 DB 업데이트 + 교사 커뮤니티 활동 |

### 매주 관계 관리 체크리스트 (ESFJ 스타일)

- [ ] 2주 이상 연락 안 한 선생님 3명에게 안부 체크인
- [ ] 기존 도입 학교 선생님 1명에게 "수업 자료/코들 활용 팁" 선제 공유
- [ ] 이번 주 미팅한 선생님 전원에게 금요일 감사 메시지
- [ ] 고객 카드(선생님 정보) 미팅 후 즉시 업데이트

### 시즌별 주간 루틴 조정

| 시기 | 주간 중점 활동 |
|------|--------------|
| 1~2월 (예산 확정기) | 신규 학교 제안/견적 집중. 연간 최대 클로징 기회 |
| 3~5월 (학기 중) | 무료 체험/파일럿 시작. 씨뿌리기 시즌 |
| 7~8월 (방학) | 교사 연수 집중 실행. 2학기 도입 의사결정 유도 |
| 9~10월 (2학기) | 사용 데이터 축적 + 성공 사례 확보. 재구매 기회 발굴 |
| 11~12월 (예산 편성) | 내년 예산 반영 활동. 교장/교감 대상 성과 보고 |

# 2팀. 국내 코들 대학 · 평생교육팀

**팀 KPI**: 매출 확대 (사업 수주 + 대학 코들 도입)

## 주간 루틴

| 요일 | 리드 (김지훈) | 팀원 (손해성) |
|------|-------------|-------------|
| **월** | 대학별 진행 현황 리뷰 + 이번 주 우선순위 딜 3개 선정 + 손해성 님 업무 방향 확인 | 주간 리서치 계획 정리 + 미팅 전 대학 정보 사전 조사 착수 |
| **화** | 교수님/사업단장 미팅 + 제안서 뼈대 작업 | 대학별 사업 현황/경쟁사 리서치 + 제안서 초안 작업 + **화요일 보고** |
| **수** | 기존 교수님 관계 너처링 (트렌드 자료 공유, 안부) + 제안서 검토 | 미팅 자료 준비 + 파일럿 운영 + 교수님 일상 연락 |
| **목** | 대학 현장 방문/미팅 + 파이프라인 점검 + **목요일 보고 (대표에게)** | 후속 자료 발송 + 제안서 수정 + **목요일 보고** |
| **금** | 다음 주 전략 수립 + 손해성 님 제안서 최종 검토 | 사업 공고 모니터링 + 시장 데이터 정리 + CRM 업데이트 |

### 매주 관계 관리 체크리스트 (ESFJ 스타일)

- [ ] 진행 중인 대학 교수님 2명에게 "도움 될 자료" 선제 공유 (교육 트렌드, 사업 동향 등)
- [ ] 파일럿 운영 중인 대학에 "중간 데이터 리포트" 발송 (교수님이 내부 보고에 활용할 수 있게)
- [ ] 사업 선정/수주 소식 있는 대학에 즉시 축하 + 지원 제안 메시지
- [ ] 고객 카드(교수님 정보 — 연구 분야, 관심사, 고민) 미팅 후 업데이트

### 대학 영업 시즌별 중점

| 시기 | 주간 중점 활동 |
|------|--------------|
| 1~3월 (사업 공고 시즌) | 공고 분석 자료 선제 제공 + 제안서 집중 |
| 4~6월 (학기 중) | 파일럿 운영 + 수업 데이터 수집 + 다음 사업 준비 |
| 7~9월 (방학~2학기) | 성과 데이터 정리 + 2학기 도입 제안 + 신규 대학 컨택 확대 |
| 10~12월 (예산 편성) | 내년 예산 반영 활동 + 기존 대학 재계약 + 성과 보고 |

# 3팀. 국내 사업 운영팀

**팀 KPI**: 매출 확대 (사업 수주 + 재수주)

## 주간 루틴

| 요일 | 리드 (최형관) | 팀원 (권민지) |
|------|-------------|-------------|
| **월** | 사업별 진행률 점검 + 이번 주 핵심 목표 3개 설정 + 권민지 님 업무 배분 (성장형 50% + 운영 50%) | 주간 업무 수령 + 이번 주 자율 기획 과제 주제 선정 |
| **화** | 외부 기관/발주처 미팅 + 네트워크 관리 | 사업 현장 운영 + 교육 콘텐츠 준비 |
| **수** | 사업 진행률 보고 (대표에게) + 이슈 해결 방안 직접 제시 | 교원 연수 운영 실행 / 사업 운영 실행 |
| **목** | 신규 사업 기회 탐색 + 제안서/입찰 서류 작업 + 권민지 님 자율 기획 중간 피드백 | 만족도/성과 데이터 수집 + 네트워크 일상 연락 |

### 매주 관계 관리 체크리스트 (ESFJ 스타일)

# 마지막으로

**루틴은 의지가 아니라 시스템이다.** 의지력으로 매주 실행하려 하면 3주도 못 간다. 이 문서의 루틴이 실제로 작동하려면:

1. **월요일 킥오프에서 이번 주 루틴을 소리 내어 확인한다**
2. **금요일 마감에서 이번 주 루틴 실행률을 숫자로 체크한다**
3. **루틴을 지키지 못한 주가 있으면, 왜 못 지켰는지 원인을 파악하고 구조를 개선한다** (의지력 탓하지 않기)

"매주 이것만 지키면 성과는 따라온다"는 확신을 팀 전체가 갖는 것. 그것이 이 문서의 목표입니다.
